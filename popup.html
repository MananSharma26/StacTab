Tab groups
What each button does
Auto-Pilot
●
Auto-Pilot toggle
On by default. Automatically groups your tabs by domain as you browse — no clicks needed. When 2+ tabs from the same site are open, they're grouped and colour-coded instantly.
#
Group at 2 / 3 / 4 / 5
The threshold before Auto-Pilot creates a group. Default is 2 — meaning 2 tabs from the same domain triggers grouping. Raise it if you want more tabs before groups appear.
Organize
✦
Organize tabs into groups
Groups all ungrouped tabs by domain. Use when Auto-Pilot is off, or to force a re-group.
Tools
☾
Sleep tabs
Suspends all background tabs to free up RAM. The tabs stay visible — click one to wake it.
⦻
Dedupe tabs
Closes duplicate tabs. Keeps the first copy of any URL that's open more than once.
⊞
Merge windows
Pulls all tabs from other Chrome windows into this one.
Archive
⊙
Current tab
Saves the active tab to your Vault and closes it.
⟷
All tabs
Saves all open tabs as a named session in your Vault, then closes them.
⟵
Left tabs
Saves all tabs to the left of the current one to your Vault and closes them.
⟶
Right tabs
Saves all tabs to the right of the current one to your Vault and closes them.
Tab groups
●
Group buttons
Each button is an active tab group. Click to archive the whole group to your Vault.
